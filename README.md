# cuevana-downloader
Simple script for download from some cuevana clone pages.

### Usage
`cuevana3-download.py URL`

### Example
`cuevana3-download.py https://www3.cuevana3.ai/40300/godzilla-vs-kong`

### Python3 libraries

- `pip3 install requests`
- `pip3 install httpx`
- `pip3 install lxml`
- `pip3 install tqdm`
- `pip3 install cssselect`
